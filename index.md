## Уровень 0

### Примеры
Для вывода в консоль используется следующая команда
```javascript
console.log("Привет")
console.log(5)
console.log(2 + 2)
console.log(1 + 10 * 10)
console.log("Привет" + "Леша")
console.log("Пять плюс пять будет " + (5 + 5))
console.log("Пять плюс пять не будет " + 5 + 5)
```

## Уровень 1

### Примеры
Для ввода данных от пользователя можно использовать следующую команду
```javascript
var a = window.prompt()
var b = window.prompt()
```

### Зачачи
#### Задача 1
Попросить пользователя ввести строку. Вывести в консоль эту строку 1 раз.
#### Задача 2
Попросить пользователя ввести строку. Вывести в консоль эту строку 3 раза.
#### Задача 3
Попросить пользователя ввести 2 строки. Вывести в консоль 2 строки введенные пользователем.
Пример:
- Пользователь вводит "Привет"
- Пользователь вводит "мир"
- Программа должна вывести

```
Привет
мир
```

#### Задача 4
Попросить пользователя ввести 2 строки. Вывести в консоль 2 строки введенные пользователем, но в обратном порядке.
Пример:
- Пользователь вводит "Привет"
- Пользователь вводит "мир"
- Программа должна вывести
```
мир
Привет
```

#### Задача 5
Попросить пользователя ввести 2 строки. Вывести в консоль 1 строку состоящую из двух строк введенных пользователем.
Пример:
- Пользователь вводит "Привет"
- Пользователь вводит "мир"
- Программа должна вывести
```
Приветмир
```

#### Задача 6
Попросить пользователя ввести 2 строки. Вывести в консоль 1 строку состоящую из двух строк введенных пользователем в обратном порядке.
Пример:
- Пользователь вводит "Привет"
- Пользователь вводит "мир"
- Программа должна вывести
```
мирПривет
```

### Примеры

```javascript
var i = 0;
while (i < 10) {
    conosle.log(i)
    i = i + 1
}
```

#### Задача 1
Вывести в консоль все числа от 1 до 100 (включительно)

#### Задача 2
Вывести в консоль все нечетные числа от 1 до 99 (включительно)

#### Задача 3
Вывести в консоль все четные числа от 0 до 100 (включительно)

#### Задача 4
Вывести в консоль все числа от 100 до 1.
Программа должна вывести
```
100
99
98
...
3
2
1
```

#### Задача 5
Вывести в консоль все числа от -1 до -100 (включительно)

#### Задача 6
Вывести в консоль квадраты всех чисел от 1 до 10. Квадрат числа 3, например, это 3 x 3 = 9

#### Задача 7
Вывести в консоль кубы всех чисел от 1 до 10. Куб числа 3, например, это 3 x 3 x 3 = 27

#### Задача 8
Попросить пользователя ввести 1 строку (число от 1 до 100).
Вывести в консоль сумму всех чисел от 1 до введенного числа.

#### Задача 9
Попросить пользователя ввести 1 строку (число от 1 до 100).
Вывести в консоль произведение всех чисел от 1 до введенного числа.
Пример:
- Пользователь вводит "4"
- Программа должна вывести "24", тк 1 x 2 x 3 x 4 = 24

#### Задача 10
Попросить пользователя ввести 1 строку (число от 1 до 100).
Вывести в консоль среднее всех чисел от 1 до введенного числа.
Пример:
- Пользователь вводит "4"
- Программа должна вывести "2.5", тк (1 + 2 + 3 + 4) / 4 = 2.5

#### Задача 11
Попросить пользователя ввести 1 строку (число от 1 до 100).
Вывести в консоль сумму квадратов всех чисел от 1 до введенного числа.
Пример:
- Пользователь вводит "4"
- Программа должна вывести "39", тк 1*1 + 2*2 + 3*3 + 4*4 = 1 + 4 + 9 + 25 = 39
